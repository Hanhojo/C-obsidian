#### 포인터도 변수이므로 초기화하지 않으면 쓰레기값을 가진다. ####
____
- **포인터에 직접 절대주소를 대입해서는 안된다.**
```c
int *p1 = 0x123445678; // 포인터는 정수값으로 초기화하므로 컴파일 에러
```

- **메모리 주소를 직접 사용하면 실행 에러가 발생한다.**
```c
int *p2 = (int*)0x12345678;
```
