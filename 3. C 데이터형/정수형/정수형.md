**정수형으로 short형, int형, long형, longlong형이 있다.**


- 데이터형의 정확한 크기는 플랫폼에 의해서 결정된다.

- sizeof(short) <= sizeof(int) <= sizeof(long) <= sizeof(long long)

- [[정수형의 유효범위]]
____

## 정수형의 분류 ##

### 부호 있는 정수형(signed) 생략가능

short

	short int
	signed short
	signed short int

int

	signed int
	signed

long

	long int
	signed long
	signed long int

### 부호 없는 정수형(unsigned)

unsigned short

	unsigned short int

unsigned int

	unsigned

unsigned long

	unsigned long int
