***컴퓨터 시스템에서는 음수를 나타내기 위해 2의 보수를 사용한다***

1. n을 2진수로 나타낸다.
2. 각 비트에 대하여 0을 1로, 1을 0으로 반전 시킨다.
3. 결과에 1을 더한다.

### 양의 정수 7 2진 표현 방법 ###

1. 00000111(7의2진수)
2. 11111000(0과 1을 반전)
3. 11111001(-7의 2진 표현) 
	0xfffffff9(-7의 16진수 표현)


```c
#include <stdio.h>

int main()
{   
	printf(" 7 = %08x\n",7);//8문자 폭에 맞춰서 16진수를 출력하면서 빈칸에는 0을 출력//
	printf(" -7 = %08x\n", -7);
	printf("7+(-10) = %08x\n", 7 + (-10));
	
	return 0
	
	/*7 = 00000007
	-7 = fffffff9
	7+(-10) = fffffffd*/
}
```
